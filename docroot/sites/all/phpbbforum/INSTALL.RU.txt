Установка

I. Установка мода phpbbdrupal в phpBB3
Инструкция по установке находится в файле contrib/phpbbdrupal/install.xml (не точная)
1. Скопируйте содержимое каталога contrib/phpbbdrupal/root в каталог установки форума.
Для совместной работы модуля phpbbforum с форумом необходимо по крайней мере применить патч для файла functions_user.php или заменить его целиком из каталога phpbbdrupal/root.
Это позволяет избежать дублирования одинакового названия функции user_delete в Drupal и phpBB.
Кроме того, измененная функция возвращает true при успешном удалении, в то время как функция phpBB всегда возвращает false, в остальном они идентичны.
Все изменения в файлах phpBB3 не влияют на работу форума в автономном режиме.

phpbbforum install paths

Module -------------------------> Your site paths
+/phpbbforum -------------------> +/sites/all/modules/phpbbforum
                                   (Path to phpBB api file: sites/all/modules/phpbbforum/includes/)
  +/includes -------------------> +/sites/all/modules/phpbbforum/includes/ 
    phpbb_api.php ----------------> phpbb_api.php (phpBB api file name:)    
    phpbb_api_subs.php -----------> phpbb_api_subs.php
    phpbb_api_recent.php ---------> phpbb_api_recent.php
  +/contrib
    +/phpbbdrupal                    (phpBB forum root path: /home/vb/www/example/public_html/phpBB3/)
      +/root ---------------------> +/phpBB3/ 
        +/includes -----------------> +/includes
          functions_user.php ----------> functions_user.php (replace or patch)
        +/styles -------------------> +/styles

II. Установка модуля phpbbforum в Drupal

1. Скопируйте содержимое каталога phpbbforum в каталог установки модулей sites/all/modules/phpbbforum/
2. Войдите с правами администратора на сайт, включите phpBBforum модуль и profile модуль, если вы собираетесь синхронизировать профильные данные.
   Без модуля profile будут сихронизироваться только учетная информация, имя, пароль, почтовый адрес, подпись и аватара.
   Для администратора Drupal с идентификатором 1 и администратором phpBB с идентификатором 2 имя, пароль, почтовый адрес в целях безопасности не синхронизируются.
3. Войдите в ваш форум желательно как администратор с тем же именем.
4. Войдите в настройки модуля phpbbforum в Drupal phpBBforum settings.
5. Установите корневой каталог размещения форума (путь к файлу форума config.php). Нажмите Сохранить.
6. При успешном вводе параметров вы увидете имя пользователя в phpBB.
   Добейтесь успешного прохождения теста аутотентификации.
7. Установите другие настройки модуля. Нажмите Сохранить.
8. Войдите в настройки блоков и установите нужные вам блоки.
   Рекомендуется установка блока phpBBforum: Hidden authentication для обеспечения авторизации и синхронизации.
   Блок не имеет заголовка и не выводит никакой информации.
   Фунции этого блока могут быть ограничены, если вы не прошли тест аутотентификации.
9. Добавьте необходимые поля в профиле. Их имена должны совпадать с именами соответствующих полей в настройках модуля.
   Для поля profile_birthday выбирайти тип date. Вы можете использовать любое существующее поле, хранящее день рождение, например, используемое модулем birthdays.

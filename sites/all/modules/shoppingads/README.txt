Requirements
------------
This module requires Drupal 5.X.

Overview:
--------
This module adds shoppingads.com advertising to nodes of activated types.

Installation and configuration:
------------------------------
Simply extract the download package in your modules directory and 
then enable the module in 'admin/build/modules/'.

For configuration options go to
'admin/settings/shoppingads'

More information is available at:
http://www.seo-expert-blog.com/tools/cms/drupal/modules/shoppingads
